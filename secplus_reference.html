ATTACK TYPES & THREAT CATEGORIES

🎭 Social Engineering

  • Phishing — Deceptive email to steal creds
  • Spear Phishing — Targeted phishing (specific person)
  • Whaling — Phishing targeting executives (C-level)
  • Vishing — Voice/phone phishing
  • Smishing — SMS text phishing
  • Pharming — Redirect DNS to fake site
  • Pretexting — Fake scenario to extract info
  • Baiting — USB drop / free download lure
  • Tailgating/Piggybacking — Physical follow through door
  • Shoulder Surfing — Observe screen/keyboard
  • Dumpster Diving — Search trash for info
  • Quid Pro Quo — Something for something exchange
  • BEC — Business Email Compromise
  • Watering Hole — Compromise site victim visits

🌐 Network Attacks

  • DoS — Overwhelm single target
  • DDoS — Distributed, multiple sources (botnet)
  • MitM / MitB — Intercept communications
  • ARP Poisoning/Spoofing — Fake MAC→IP mapping
  • DNS Poisoning — Corrupt DNS cache
  • DNS Hijacking — Redirect DNS responses
  • IP Spoofing — Fake source IP address
  • Smurf Attack — ICMP amplification DDoS
  • Teardrop — Malformed fragmented packets
  • Ping of Death — Oversized ICMP packet
  • SYN Flood — Half-open TCP connections
  • Session Hijacking — Steal session token
  • SSL Stripping — Downgrade HTTPS to HTTP
  • Evil Twin — Rogue AP same SSID
  • Replay Attack — Resend captured auth packet
  • Deauthentication — Force 802.11 disconnect

💉 Application Attacks

  • SQL Injection (SQLi) — Inject SQL into input
  • XSS (Stored) — Persistent malicious script in DB
  • XSS (Reflected) — Script in URL/response
  • XSS (DOM) — Client-side DOM manipulation
  • CSRF/XSRF — Trick authenticated user
  • SSRF — Server-Side Request Forgery
  • XXE — XML External Entity injection
  • Path Traversal — ../../../etc/passwd
  • LFI/RFI — Local/Remote File Inclusion
  • Command Injection — OS commands via input
  • LDAP Injection — Manipulate LDAP queries
  • Buffer Overflow — Overflow memory to execute code
  • Heap Spray — Fill heap to control exec
  • Use-After-Free — Reuse freed memory
  • Race Condition — TOCTOU timing attack
  • Clickjacking — UI redress, invisible overlay
  • Open Redirect — Redirect to malicious site
  • Insecure Direct Object Ref — IDOR access control bypass

🦠 Malware Types

  • Virus — Requires host file, self-replicates
  • Worm — Self-propagates, no host needed
  • Trojan — Appears legitimate, hides malware
  • RAT — Remote Access Trojan (backdoor)
  • Rootkit — Hides itself in OS kernel
  • Bootkit — Infects MBR/bootloader
  • Ransomware — Encrypts files, demands payment
  • Spyware — Monitors user activity
  • Adware — Displays unwanted ads
  • Keylogger — Records keystrokes
  • Botnet/Bot — Remote-controlled zombie
  • Logic Bomb — Triggered by condition/date
  • Fileless Malware — Lives in memory/registry
  • PUP/PUA — Potentially Unwanted Program/App
  • Backdoor — Unauthorized remote access
  • Cryptojacker — Mines crypto using victim resources
  • Polymorphic — Changes signature to evade AV
  • Metamorphic — Rewrites its own code completely

🔐 Password / Credential Attacks

  • Brute Force — Try all combinations
  • Dictionary Attack — Wordlist of common passwords
  • Rainbow Table — Precomputed hash lookup
  • Credential Stuffing — Reuse leaked credentials
  • Password Spraying — One password → many accounts
  • Pass the Hash — Use NTLM hash without cracking
  • Pass the Ticket — Use Kerberos ticket (Golden/Silver)
  • Golden Ticket — Forge Kerberos TGT (krbtgt hash)
  • Silver Ticket — Forge Kerberos service ticket
  • Kerberoasting — Request + crack service tickets
  • AS-REP Roasting — Attack no-preauth accounts
  • Mimikatz — Extract credentials from LSASS
  • Shoulder Surfing — Visual observation
  • Birthday Attack — Hash collision probability

☁️ Cloud & Modern Threats

  • Misconfiguration — #1 cloud threat (open S3 buckets)
  • SSRF in Cloud — Access metadata API 169.254.169.254
  • Supply Chain Attack — Compromise vendor/software (SolarWinds)
  • API Abuse — Exploit API endpoints
  • Cryptojacking — Unauthorized crypto mining
  • Container Escape — Break out of Docker/K8s
  • Serverless Injection — Code injection in FaaS
  • Shadow IT — Unauthorized cloud services
  • Insider Threat — Malicious/negligent employee
  • Zero-day — Unknown/unpatched vulnerability
  • APT — Advanced Persistent Threat (nation-state)
  • Living off the Land — Use legit tools (LOLBins)
  • Lateral Movement — Move inside network post-compromise
  • Privilege Escalation — Gain higher access rights
